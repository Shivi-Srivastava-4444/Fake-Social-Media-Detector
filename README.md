# Fake-Social-Media-Detector
This is a project that helps detect whether a social media account is trustworthy or not. It gives a trust score and the reasons why it is a risky account. A simple front-end demo that detects potentially fake social media profiles using rule-based heuristics and explains why a profile looks suspicious.
